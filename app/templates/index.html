What is klotski

Klotski is a sliding block puzzle thought to have originated in the early 20th century. The name in a more global sense to refer to a whole group of similar sliding-block puzzles where the aim is to move a specific block to some predefined location.

For the original Klotski puzzle, the largest block must be moved to the bottom middle location so that it can be slid over the border, without any of the other blocks being removed in this way.

Klotski puzzle

It is still unknown which version of the puzzle is the original. However, the early cognate of Klotski closest in design dates to 1909 in Chicago.

Nowadays, there are several variations of this game. Some changes names specific to the culture of country, like Huarong Dao (China) and L'âne rouge (France); Some with different arrangement of blocks, like Pennant Puzzle and Ma's Puzzle.

How to Play

N puzzle is a sliding puzzle having n square tiles numbered 1-n in a frame, leaving one unoccupied tile position. The goal of the puzzle is to place the tiles in numerical order.
